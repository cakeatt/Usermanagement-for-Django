# Usermanagement-for-Django

    在我学习Django中写的一个简单的后台管理系统

    本项目前端运用html、css、js，并在其中加入了bootstrap及jQuery

    后端使用了python的Django框架，数据库使用的是MySQL

    运行这个项目只需要拉取代码，然后将UserManagement中的settings.py文件中的


DATABASES= {

    'default': {

    'ENGINE': 'django.db.backends.mysql',

    'NAME': '你的数据库名字,

    'USER': 'root',

    'PASSWORD': '你的数据库密码',

    'HOST': '127.0.0.1',

    'PORT': '3306',

    }

}

    即可运行，由于本人也是个新手，欢迎提出不足
